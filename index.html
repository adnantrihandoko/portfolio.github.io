Tentang Saya

Saya adalah mahasiswa Teknik Informatika dengan minat yang mendalam dalam pengembangan perangkat lunak, khususnya menjadi seorang Software Developer. Memiliki pengalaman dalam membangun aplikasi berbasis web dan aplikasi mobile, saya menguasai berbagai teknologi mulai dari pengembangan frontend menggunakan HTML, CSS, JavaScript, dan Laravel Blade hingga pengelolaan backend dengan menggunakan PHP, Laravel. Saya terus mengembangkan keterampilan saya dalam pemrograman dan sistem manajemen basis data untuk menciptakan solusi perangkat lunak yang efisien dan user-friendly.

Sebagai mahasiswa, saya berfokus pada penerapan teknologi terbaru dan penerapan praktik terbaik dalam pengembangan perangkat lunak untuk mencapai hasil yang optimal. Saya selalu berusaha untuk meningkatkan kemampuan teknis saya dan beradaptasi dengan perkembangan industri teknologi yang terus berubah. Saya berkomitmen untuk mengasah keterampilan saya, dan memiliki ambisi untuk berkontribusi pada proyek-proyek teknologi yang inovatif di masa depan.
